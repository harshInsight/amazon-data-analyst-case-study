# Tool Selection & Justification

## Data Extraction
Amazon SP-API is used as it provides official and real-time access to Seller Central data.

## Automation
Python is chosen due to its strong ecosystem, automation capabilities, and data processing libraries.

## Storage
- Raw data stored in Google Drive or local file system for backup and audit
- PostgreSQL used for structured, relational storage

## Scheduling
Cron / Windows Task Scheduler enables automated daily, weekly, and monthly jobs.

## Cost Consideration
Most tools are open-source, keeping monthly cost under $20.
