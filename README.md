# Amazon Seller Data Analyst Case Study

## Candidate
Harsh Sharma

## Overview
This repository contains the solution for the Data Analyst technical assessment provided by Menko Services Pvt. Ltd.

The case study focuses on designing a scalable data architecture, API integration, and data preprocessing & analysis for Amazon Seller data across multiple countries.

---

## Phase-wise Breakdown

### Phase 1: Logical Schema & Tool Selection
- Designed a normalized logical database schema for Orders, Payments, Fulfillment, and Finance data
- Justified tool selection with a cost-efficient approach
- Schema image and explanation included

### Phase 2: API Integration
- Python code demonstrating authentication and API calls
- No real credentials used
- Focus on structure, best practices, and scalability

### Phase 3: Data Preprocessing & Analysis
- Python-based preprocessing of Excel files
- Date standardization, null handling, duplicate removal
- Currency conversion to EURO
- Consolidation of multi-country datasets
- Sample analytical queries
